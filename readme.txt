1. install required libraries using following command
	pip install tensorflow,keras,pillow,numpy,pandas,flask,os
2. Download the folder named as Traffic Sign Classifier
3. Run the command going to the location of Traffic Sign Classifier folder: python traffic_sign_classifier.py
4. Open browser and open link http://127.0.0.1/4555/
5. Upload the image and get the output
